Shorturl

This is to create shorter URL's for use in documentation and on PCB's. There are some limitations to this URL shortener, it is only fired on the MainPage handler as such this can not be used on any other handlers. The path can thus be anything that is not a valid handler.

Usage:

  • Path:

    Do not start with a slash, any path is allowed as long as it is not a valid handler.

  • HTTP Code:

    The HTTP code to send, this can be one of these values:

    • 500: Internal Server Error
    • 404: File Not Found
    • 403: Access Denied
    • 302: Temporary Redirect
    • 301: Permanent Redirect (Most common)

  • Location:

    Only required for HTTP error codes 302, and 301.
